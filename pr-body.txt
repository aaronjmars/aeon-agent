## Summary

- **New skill** `skills/rss-feed/SKILL.md` — scans `articles/*.md`, runs the generator script, commits the updated feed, and sends a notification with the subscribe URL
- **New script** `scripts/generate-feed.sh` — standalone bash script that parses markdown articles (extracts title from first `#` heading, date from filename pattern `YYYY-MM-DD`, first paragraph as summary) and outputs valid Atom XML to `articles/feed.xml`. Handles frontmatter, XML escaping, and auto-detects the repo slug from git remote
- **Updated `aeon.yml`** — adds `rss-feed` skill scheduled at 5:30 PM UTC (after content-generating skills finish), disabled by default
- **Updated `README.md`** — adds `rss-feed` to the Meta/Agent skills table and a new RSS Feed section with the subscribe URL

## Technical details

- Valid Atom 1.0 XML with proper feed, entry, title, summary, updated, link, and id elements
- Links point to github.com blob URLs for reading, raw.githubusercontent.com for feed self-link
- Date extraction: tries YYYY-MM-DD pattern in filename first, falls back to git commit date, then file mtime
- Title extraction: skips YAML frontmatter, finds first heading
- Summary extraction: first non-empty, non-heading line after the title
- All text is XML-escaped (ampersands, angle brackets, quotes)
- Script is idempotent — safe to run on every cron tick
